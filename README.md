# Company Network Lab

Production-style AWS network built with Terraform.

## What this builds
- 1 VPC
- 2 public subnets
- 2 private app subnets
- 2 private DB subnets
- Internet Gateway
- NAT Gateway
- Public and private route tables
- Security groups for ALB, app tier, and DB tier

## Skills demonstrated
- AWS networking
- Terraform infrastructure as code
- CIDR planning
- Public/private subnet design
- Routing and NAT
- Security group design

## Commands used
terraform fmt
terraform init
terraform validate
terraform plan
terraform apply
terraform destroy
